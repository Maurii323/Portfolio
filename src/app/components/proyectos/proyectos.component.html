Proyectos

Gestion de Tareas

Administrador de tareas en Django con funcionalidades completas de CRUD, registro y login de usuarios. Utilizacion de bootstrap para los estilos. SQLite3 como base de datos en desarrollo y PostgreSQL en produccion(alojada en Neon). Desplegado en Render, ya que esta con el plan free puede tardar 1 minuto en cargar.

Torneos de Futbol

Administrador de torneos de fultbol en Django. Utilizacion de bootstrap para los estilos. SQLite3 como base de datos en desarrollo y PostgreSQL en produccion(alojada en Neon). Desplegado en Render, ya que esta con el plan free puede tardar 1 minuto en cargar.


BackEnd Finanzas

Api del proyecto finanzas desarrollado en Django Rest Framework, con funcionalidades de autenticacion con JWT(access y refresh token) y CRUD de transacciones y sus categorias. SQLite3 como base de datos en desarrollo y PostgreSQL en produccion(alojada en Neon). Uso de Postman para testear los endpoints y render para alojar la api.

FrontEnd Finanzas

Front desarrollado en Angular que consume la api "BackEnd Finanzas", con funcionalidades de autenticacion con JWT(access y refresh token) y CRUD de transacciones y sus categorias. Luego de hacer la primera consulta a la api(registrarse o loguearse) tardara 1 minuto debido al plan gratuito de render
